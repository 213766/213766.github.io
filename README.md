Blog pessoas desenvolvido para a matéria de Desenvolvimento Web 2026/01

Enunciado do trabalho:
1. Objetivo
O objetivo deste trabalho é aplicar os conceitos fundamentais de HTML5, CSS3 e a introdução à lógica com JavaScript, focando na organização de arquivos (estrutura de pastas), semântica e navegação entre páginas.
2. Requisitos Técnicos
A entrega deve seguir obrigatoriamente a estrutura de pastas padrão para projetos web, garantindo a organização do código:
• Raiz do projeto: Arquivos HTML (ex: index.html).
• /css: Arquivos de folha de estilo (style.css).
• /js: Arquivos de script (script.js).
• /assets: Imagens, ícones e mídias utilizados no projeto.
3. Estrutura de Páginas
O blog deve ser composto por, no mínimo, quatro páginas principais interconectadas por um menu de navegação persistente:
1. Home (index.html):
• Página inicial com uma breve introdução ao blog.
• Uma seção de “Últimas Postagens” (pode ser estática, simulando um feed).
2. Sobre Mim (about.html):
• Texto narrativo sobre sua trajetória acadêmica, profissional e interesses.
• Pelo menos uma imagem representativa (armazenada na pasta /assets).
3. Currículo (curriculum.html):
• Utilizar tags de lista (<ul>, <li>) para listar experiências.
• Uso de tabelas (<table>) para organizar formação acadêmica ou competências técnicas.
4. Contato (contact.html):
• Página contendo um formulário estruturado para contato.
5. Receitas Favoritas (receitas.html):
• Uma lista de receitas de sua preferência.
• Obrigatório: Criar uma subpágina específica para Mousse de Maracujá,
detalhando ingredientes e modo de preparo de forma bem estruturada semanticamente.
4. Requisitos de Estilização (CSS)
A aparência do projeto deve demonstrar maturidade visual:
• Uso de CSS Externo (arquivo isolado na pasta específica).
• Identidade visual consistente: mesma paleta de cores e tipografia aplicadas a todas
as páginas.
• Uso de modelos de layout modernos como Flexbox ou CSS Grid para organização
dos elementos.
5. Requisitos de Interatividade (JavaScript)
É obrigatório o uso de um script externo para as seguintes funcionalidades:
• Modo Noturno (Dark Mode): Implementar um botão que alterne as cores de
fundo e de texto da página.
• Validação de Formulário: Verificação simples dos campos no formulário de
contato.
• Acessibilidade: Botão que permita aumentar/diminuir o tamanho da fonte das
páginas para facilitar a leitura.
